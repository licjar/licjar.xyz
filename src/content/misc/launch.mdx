HTMLは木構造をしているが、人間はユーザーに木以外の構造を見せたいので、木の上にアノテーションをして木構造を壊そうとする、そこに本質的矛盾があると思った。

手元で`git push`とやると Cloudflare のサーバーでビルドが行われて1分とかのうちにスマホから見れるようになるのは不思議なことだと感じた。<a href="https://hsjoihs.hatenablog.com/">hsjoihs</a>「なぜ half the internet が Cloudflare なのかがわかりますね〜」

不思議というか、レイヤーが短絡していて怖いというのが近いかも。自宅PCのターミナル操作が Cloudflare のサーバーに直結するのはちょっと怖い。人は Docker と Cloudflare によってなにもしなくてよくなったんだなぁ
